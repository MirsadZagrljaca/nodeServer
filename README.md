<h1>My First NodeJS Server</h1><br/>

Start project with: <b>npm start</b><br/>
Move project to production mode with: <b>npm run build</b><br/>
<br/>
<h3>Packages used:</h3>
  <h4>Production mode:</h4>
    express<br/>
    body-parser<br/>
    helmet<br/>
  <h4>Development mode:</h4>
    nodemon<br/>
    webpack<br/>
    babel-core<br/>
    babel-loader<br/>
    babel-preset-env<br/>
    babel-preset-stage-2<br/>
<h4>Note: Every package used is latest version expect for babel-loader which is 7.1.5</h4>
